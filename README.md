# システム設計2022


# 必要なもの
* xampp 3.3.0
* sqlite3 3.39.3
* ngrok
* composer
* twig
* ramesy/uuid
* phpdotenv

# 環境設定の流れ
- xamppのインストール
- composerのインストール
   - twig
   - ramsey/uuid
   - phpdotenv
- sqliteのインストール
- プログラムのインストール
- slack apiの設定
- ngrokのインストール
- タイムスケジューラの設定
